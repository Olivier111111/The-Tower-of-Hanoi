# The-Tower-of-Hanoi

## Overview
This is a Unity-based simulation of the Towers of Hanoi puzzle. It allows players to visualize and experiment with different solution strategies, including classical, weighted, and cyclic variations. The project supports both 3-peg and 4-peg setups.

## Features
- Classical, weighted, and cyclic versions of Towers of Hanoi.
- Dynamic spawning of disks and pegs based on user selection.
- Adjustable number of disks and pegs.
- Interactive gameplay: drag and drop disks onto pegs.
- Visual demonstration of how different rules affect solutions.

## How to Play
- Open the Unity project in Unity 2021 or newer.
- Run the Menu Scene.
- Use the dropdowns to select:
* Number of pegs (3 or 4)
* Number of disks (1â€“5)
* Game variation (Classical, Weighted, Cyclic)
- Click Start Game to load the puzzle scene.
- Drag and drop disks to solve the puzzle according to the rules of the selected variation.

## Requirements
- Unity 2021 or newer
- C# support (standard Unity setup)

## Notes
Disk weights in the weighted variation can be set manually in the Inspector.
